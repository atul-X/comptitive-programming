# Competitive Programming Repository

A well-organized collection of competitive programming solutions and algorithm implementations in Java.

## 🎯 Purpose

This repository serves as a comprehensive resource for competitive programming practice, containing solutions to various algorithmic problems organized by difficulty, topic, and contest platform.

## 📁 Structure Overview

The codebase is organized into four main categories:

- **🔧 Algorithms**: Implementations of various algorithms by category (48+ files)
- **📊 Data Structures**: Core data structure implementations (5 files)
- **🎯 Practice**: Problems organized by difficulty level
- **🏆 Contest**: Solutions from different competitive programming platforms

### Recent Improvements
- ✅ Renamed files with proper naming conventions (PascalCase)
- ✅ Organized files into correct algorithm categories
- ✅ Removed duplicate and low-quality code
- ✅ Fixed compilation errors and package declarations
- ✅ Cleaned up directory structure

## 🚀 Quick Start

### Running a Solution
```bash
# Navigate to any solution directory
cd src/algorithms/array/

# Compile and run
javac ArrayConcatenation.java
java ArrayConcatenation
```

### Package Structure
All Java files follow the package naming convention:
- `algorithms.category` for algorithm implementations
- `datastructures` for data structure code
- `practice.level` for practice problems
- `contest.platform` for contest solutions
